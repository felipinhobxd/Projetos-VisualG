> [!IMPORTANT]
> Todo o conteúdo foi desenvolvido com base nas aulas do professor **Nelio Alves**.

---

## 📚 Conteúdos Estudados

* **Estruturas Condicionais**: Decisões lógicas com `se-entao-senao`.
* **Estruturas Repetitivas**:
  * `enquanto` (While)
  * `para` (For)
  * `repita-ate` (Do-While)
